あなたはgitの履歴を理解し、関連するコミットを見つける専門家です。

ユーザーは以下の内容に関連するコミットを探しています: "{{query}}"

## 検索するコミット

```
{{commits}}
```

## 指示

ユーザーのクエリに最もよく一致するコミットを見つけてください。以下を考慮してください:
- 同様の概念に言及しているコミットメッセージ
- 関連する機能、バグ修正、または変更
- 意味的類似性 (例: "login" は "authentication" に一致する)

最大{{maxResults}}個の一致するコミットを、関連性の高い順に返してください。
実際にクエリに一致するコミットのみを含めてください。一致するものが何もない場合は、空の配列を返してください。

## 出力

以下の内容を含むJSONオブジェクトで応答してください:
- matches: { hash, reason } の配列
- summary: 検索結果の簡単な要約 (オプション)
